◆ read_path_waypoints()

pathfinder_group3.read_path_waypoints ( csv_file = 'path.csv')

Reads path waypoints from MATLAB CSV and returns a simplified sequence.

Sequence contains only turning points and start/end points. Can only detect new obstacles at corners and will miss obstacles in straight paths.

Parameters
csv_filePath to the CSV file containing waypoint data
Returns
List of normalized waypoint tuples (x, y, z)

◆ run_sequence()

pathfinder_group3.run_sequence ( scf,
sequence,
base_x,
base_y,
base_z,
yaw )

Flies the drone through a given sequence of positions.

Checks for obstacles at each position and logs them if detected. Sets starting point as (0,0,0) and moves relative to that point instead of global lighthouse coordinates. Takes approximately 0.5 seconds to move from point to point.

Parameters
scfSyncCrazyflie object
sequenceList of position tuples to fly through
base_xBase X coordinate offset
base_yBase Y coordinate offset
base_zBase Z coordinate offset
yawYaw angle in degrees
